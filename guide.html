Guide

How to get a figure that holds up in print, and a short primer for anyone new to SMILES.

Drawing one structure

  1. Enter the compound. Type a name and press Enter to look it up on PubChem, or paste a SMILES string. SMILES is faster and needs no connection once the page has loaded.
  2. Check the numbers. The strip below the structure shows molecular weight, heavy atom count, and ring count. If they are not what you expect, the input was not the molecule you meant.
  3. Pick a format. SVG for anything going into a manuscript, PNG when a system will not accept vector files.
  4. Download. The file is written straight from your browser to your computer.

SVG or PNG

SVG stores the drawing as instructions rather than pixels. It stays sharp at any size and usually weighs a few kilobytes, which makes it the right default for journals, theses, and LaTeX.

PNG is a fixed grid of pixels. Use it when something downstream cannot accept a vector file, such as certain submission portals and slide templates.

Backgrounds

Downloads are white by default, which is what journals and slide decks expect.

Turn on transparency only when you are placing the structure over a colored panel or a photograph. Two warnings: some manuscript systems flatten transparency to black rather than white, and Word has a long history of mishandling transparent PNGs when printing. If the figure has to work everywhere, leave transparency off.

Before you submit, open the downloaded file on its own at full size. Depictions that look fine in a small preview often show overlapping labels once they are enlarged.

Building a panel figure

The panel builder puts several structures into one image, which is how most medicinal chemistry figures are laid out. One compound per line, in either form:

Aspirin | CC(=O)OC1=CC=CC=C1C(=O)O
Caffeine | CN1C=NC2=C1C(=O)N(C)C(=O)N2C
Warfarin

A line with a label and a SMILES string draws immediately. A line with only a name is looked up on PubChem first, so it needs a connection and takes a moment longer. The column setting controls the grid shape, and the export is one figure rather than separate files.

A short SMILES primer

SMILES writes a structure as a line of text. These rules cover most drug-like molecules:

  • Atoms are element symbols. Carbon chains are written CCC, and hydrogens are assumed.
  • Bonds are single by default, = for double, # for triple.
  • Branches go in parentheses, so CC(C)C is isobutane.
  • Rings open and close with a matching digit, so C1CCCCC1 is cyclohexane.
  • Aromatic rings use lowercase, so c1ccccc1 is benzene.
  • Charges and isotopes go in brackets, as in [NH4+] and [13C].
  • Stereochemistry uses / and \ for double bonds, @ and @@ for chiral centers.

If you would rather not write one by hand, look the compound up by name here, or copy the canonical SMILES from its PubChem page.

If something goes wrong

MessageWhat it usually means
SMILES could not be readAn unclosed ring digit, an unbalanced bracket, or a capital C where an aromatic lowercase c belongs.
PubChem has no compound by that nameTry the generic or systematic name rather than a brand name, check the spelling, or paste SMILES instead.
The drawing engine could not loadThe page fetches RDKit on first load. Reload with a working connection. Some university firewalls block this.
Lookups fail when you build a large panelPubChem limits how fast requests can arrive. Give SMILES directly to avoid the limit.

Privacy

Structures are drawn inside your browser. Nothing you type is sent to a server and nothing is stored. The exception is name lookup, which has to send the name you typed to PubChem. If a structure is confidential, paste it as SMILES rather than looking it up.

Doing this in Python

For batch work over CSV files, scripted pipelines, or automatic captions, use the MolPic Python package:

pip install git+https://github.com/akinwumiishola5000/MolPic.git
molpic generate "aspirin" -o aspirin.svg --no-h --name "Aspirin"
