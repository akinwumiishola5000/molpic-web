Downloads have a white background

Both formats export on white by default, which is what journals and slide decks expect. Switch on transparency only if you're compositing over a coloured layout.

SVG scales without limit

Vector output stays sharp at any size, so it survives resizing in Word, Illustrator, or LaTeX. Reach for PNG only when a system refuses vector files.

Many structures at once

The panel builder lays out a labelled multi-structure figure in a grid — the same job as molpic grid in the Python package.
