About MolPic Web

A browser front end for MolPic, an open-source tool for turning compound names and SMILES strings into clean molecular figures.

Why it exists

Getting a decent picture of a molecule is a small task that is oddly annoying to do well. Free depiction sites tend to be cluttered with adverts, cap resolution unless you register, or export images at a size that falls apart in print. Proper cheminformatics software solves the problem properly but requires an install and a working Python environment, which is a real barrier for a student, a reviewer, or anyone opening a laptop between meetings.

MolPic Web sits in that gap: a page you can open, use once, and close, with output good enough to put in a manuscript.

How it relates to the Python package

The MolPic Python package is the fuller tool — batch processing from CSV, automatic caption files, hydrogen suppression, reproducible scripting, and a Colab-first workflow. This site covers the part people need most often and most urgently, and asks nothing of them in return.

Use the website when

You need one figure or a small panel now, you're on a machine without Python, or you're showing someone a structure.

Use the package when

You're processing hundreds of compounds, scripting a reproducible pipeline, or need captions generated alongside figures.

Both give the same depictions

RDKit does the drawing in each case, so a structure looks the same whichever route you take.

How it stays free

There is no server. The page is static files served by GitHub Pages, and the chemistry runs as WebAssembly in your browser. There is no database to pay for, no compute to bill, no account system, and nothing to monetise. That is a deliberate architectural choice rather than a promise — the running cost is zero, so it can stay free indefinitely.

Built on

  • RDKit.js — the RDKit cheminformatics toolkit compiled to WebAssembly, which parses structures and renders every depiction.
  • PubChem — the NIH chemical database, used to resolve compound names to structures.

Both are freely available to the research community, and this tool would not be possible without them.

Citing MolPic

If MolPic contributed to work you're publishing, please cite the archived release:

Akinwumi, I. A. MolPic: Name/SMILES to publication-ready
molecular figures. Zenodo. https://doi.org/10.5281/zenodo.18089336

A CITATION.cff and BibTeX entry are included in the repository.

Reporting a problem

If a structure draws incorrectly, a name fails to resolve, or something breaks in your browser, please open an issue on GitHub. Including the SMILES string and your browser version makes it far quicker to reproduce.
