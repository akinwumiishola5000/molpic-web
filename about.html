The project

About

MolPic turns compound names and SMILES strings into clean molecular figures. This site is the version that runs in a browser.

Why it exists

Getting a good picture of a molecule is a small task that is harder than it should be. Many free sites carry advertising, limit resolution unless you register, or produce images that fall apart in print. Full cheminformatics software solves the problem properly, but it needs an install and a working Python environment, which is a real barrier for a student or a reviewer with a laptop and ten minutes.

This site covers the case where you need one figure now.

The Python package

The MolPic Python package is the fuller tool. It handles batch processing from CSV files, generates caption files, suppresses explicit hydrogens, and runs in scripts and Google Colab.

Use this site when

You need one figure or a small panel, you are working on a machine without Python, or you are showing someone a structure.

Use the package when

You are processing hundreds of compounds, scripting a pipeline you need to reproduce, or generating captions alongside figures.

Same drawings either way

RDKit does the rendering in both, so a structure looks the same whichever route you take.

How it works

The site is plain HTML and JavaScript with no server behind it. RDKit.js, the RDKit toolkit compiled to WebAssembly, parses each structure and produces every drawing inside your browser. Compound names are resolved through the PubChem API at the NIH. Both are freely available to researchers, and this tool would not exist without them.

Citing MolPic

If MolPic contributed to published work, please cite it:

Akinwumi, I. A. MolPic: Name/SMILES to Publication-Ready
Molecular Figures. ChemRxiv, 2026.
doi:10.26434/chemrxiv-2026-wjnk9

The software release is archived separately at doi:10.5281/zenodo.18089336, and a CITATION.cff file is included in the repository.

Reporting a problem

If a structure draws incorrectly or a name fails to resolve, please open an issue on GitHub. Including the SMILES string and your browser version makes it much easier to reproduce.
